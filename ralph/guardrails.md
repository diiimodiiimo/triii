# Guardrails (Append-only)

### Guardrail 1: Avoid premature complexity
- Trigger: Suggesting new tools, frameworks, or infra
- Instruction: Default to existing Triii structure unless blocked

### Guardrail 2: One ship track at a time
- Trigger: Planning work across multiple products in the same day
- Instruction: Only one project gets "ship" priority; others get maintenance-only tasks

### Guardrail 3: Files over chat
- Trigger: Long explanations in chat without writing output
- Instruction: Write decisions, plans, or conclusions to markdown immediately
